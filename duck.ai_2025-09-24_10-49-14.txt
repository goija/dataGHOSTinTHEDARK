Prima. Hier zijn concrete stappen per platform.

Firefox (desktop, Linux/Windows/macOS)
1) WebRTC uitschakelen
- Adresbalk: about:config
- Zoek: media.peerconnection.enabled → zet op false
- Optional: media.navigator.enabled → false

2) Fingerprinting minimaliseren
- about:config → privacy.resistFingerprinting → true
- about:preferences → Privacy & Security → Enhanced Tracking Protection → Strict
- about:preferences → HTTPS-Only Mode → Enable

3) Referer beperken
- about:config:
  - network.http.referer.XOriginPolicy → 2
  - network.http.referer.XOriginTrimmingPolicy → 2

4) Add-ons
- uBlock Origin (basis + “uBlock Filters – Privacy” aan)
- CanvasBlocker of NoScript (optioneel, kan sites breken)

5) DNS-over-HTTPS
- about:preferences → Network Settings → Enable DNS over HTTPS → Provider: Cloudflare of NextDNS

IPv6
- Als je VPN IPv6 niet ondersteunt: schakel IPv6 uit op OS of in de VPN-client. Anders: laat IPv6 aan zodat je geen “leak” hebt buiten VPN.
- Test: https://ipleak.net of https://browserleaks.com/ip

Linux (NetworkManager)
- Settings → Network → jouw interface → IPv6 → Disable
- Of sysctl:
  - sudo sh -c 'printf "net.ipv6.conf.all.disable_ipv6=1\nnet.ipv6.conf.default.disable_ipv6=1\n" >/etc/sysctl.d/99-disable-ipv6.conf'
  - sudo sysctl --system
- Terug aanzetten: zet waarden op 0 en herlaad.

Windows 10/11
- Control Panel → Network and Sharing Center → Change adapter settings → rechterklik adapter → Properties → vink “Internet Protocol Version 6 (TCP/IPv6)” uit.
- PowerShell (admin) uit:
  - Disable-NetAdapterBinding -Name "*" -ComponentID ms_tcpip6
- Aan:
  - Enable-NetAdapterBinding -Name "*" -ComponentID ms_tcpip6

Android
- Stock Android heeft geen globale toggle; gebruik een VPN met IPv6-ondersteuning (bv. Mullvad, IVPN).
- Per wifi-netwerk: Lang drukken op netwerk → Modify → Advanced → IP settings → IPv6 uit (alleen beschikbaar op sommige ROMs).
- Firefox Android: about:config → media.peerconnection.enabled → false

WebRTC in Chrome/Chromium (als je die gebruikt)
- Installeer “WebRTC Network Limiter” of “uBlock Origin” + blokkades; volledige WebRTC-uit is lastig. Overweeg Firefox als privacyprioriteit.

Extra netwerk-hardening
- Router: schakel IPv6 uit op WAN als je het niet gebruikt, of configureer het correct (SLAAC/DHCPv6).
- DoH/DoT via router of app (NextDNS, AdGuard).

Snelcheck na wijzigingen
- IP/DNS/WebRTC: browserleaks.com/ip en browserleaks.com/webrtc
- IPv6: test-ipv6.com
